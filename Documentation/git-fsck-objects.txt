git-fsck-objects(1)
===================

NAME
----
git-fsck-objects - Verifies the connectivity and validity of the objects in the database


SYNOPSIS
--------
[verse]
'git fsck-objects' ...

DESCRIPTION
-----------

This is a synonym for linkgit:git-fsck[1].  Please refer to the
documentation of that command.
